# bpdf

## 概要
bpdfは、指定したテキストからPDFファイルを生成するPythonクライアントです。シンプルなAPIでテキストをPDFに変換し、フォントのカスタマイズも可能です。

## 使い方
1. フォントファイル（例: `font/Lora-Regular.ttf`）をプロジェクトの`font`ディレクトリに配置します。
2. コマンドラインから以下のように実行してテキストをPDFに変換します。

```bash
python main.py "ここにPDFにしたいテキストを入力"
```

3. 実行後、`output/output.pdf`にPDFファイルが生成されます。

## フォントファイルについての注意
- `font/Lora-Regular.ttf`が存在しない場合、デフォルトのArialフォントが使用されます。
